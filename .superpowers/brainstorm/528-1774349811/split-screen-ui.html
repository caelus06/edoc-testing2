How it works step by step

1. Page Load
  • Fetches request + student data from DB
  • Loads uploaded PDFs from request_files
  • Left panel: first PDF in <iframe>
  • Dropdown to switch between files
2. OCR Extraction
  • Runs on page load (or "Re-run OCR" button)
  • Tesseract extracts raw text from PDF
  • Regex patterns parse: name, ID, course, date
  • Fields pre-filled + confidence indicators
3. Registrar Review
  • Reviews extracted fields against source PDF
  • Corrects any OCR errors (e.g., "0" → "o")
  • Fills in non-OCR fields (date issued, registrar)
  • SO number auto-suggested, editable
4. Generate & Download
  • "Confirm & Generate" → creates DOCX via PHPWord
  • Record saved to school_orders table
  • Redirects to download page with options:
  • Download DOCX | Download as PDF
